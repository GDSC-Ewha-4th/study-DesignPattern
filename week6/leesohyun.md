커맨드 객체를 안쓰면 기능 요청할 때 메소드형태로 많이 씀
그렇게 하면 외부에서 볼 때 어떤 역할인지 아는데
커맨드 객체를 쓰면 execute()라는 큰 메소드 안에서 매개변수를 호출 명령을 내리면 execute라는 명령을 내리면 캡슐화
요청 내역을 객체로 캡슐화해서 보낼 수 있음
커맨드 객체가 캡슐화한거고 invoke 형태로 쓰는게 매개변수
